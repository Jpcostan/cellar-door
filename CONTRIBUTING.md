# Contributing

Thanks for helping improve cellar-door.

## Ground rules
- Keep defaults safe (no silent permission widening).
- Add tests for new behavior.
- Update docs when user-facing behavior changes.

## Development

```bash
npm install
npm run lint
npm run typecheck
npm run test
npm run build
npm run smoke
```

## Reporting issues

Include:
- what you expected
- what happened
- repro steps
- environment (OS, Node version)
